## RF-LEG-05 – Generar reporte de auditoría

### Descripción
El sistema debe permitir al área jurídica generar reportes descargables con la información auditada de operaciones relacionadas a un trader.

### Actor(es)
- Usuario Área Jurídica

### Precondiciones
- Usuario autenticado.

### Flujo básico

1. El usuario selecciona un trader o conjunto de operaciones.
2. El sistema genera un reporte con la información auditada.
3. El sistema permite descargar el reporte.

### Información mínima del reporte

- datos del trader
- datos del comisionista
- listado de operaciones
- fechas de operación
- acciones involucradas
- cantidades
- precios
- estado del contrato.

### Postcondiciones

- El reporte queda disponible para descarga.
