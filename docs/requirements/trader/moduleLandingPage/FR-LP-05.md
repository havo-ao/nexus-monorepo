# RF-LP-05 – Visualizar información sobre comisionistas

### Descripción  
El sistema debe permitir al usuario visualizar información general sobre el servicio de comisionistas, incluyendo su rol y beneficios dentro de la plataforma.

### Actor(es)  
- Trader (no autenticado)

### Precondiciones  
- El usuario accede a la landing page  

### Flujo básico  
1. El sistema presenta una sección informativa sobre comisionistas  
2. Se explica su rol como intermediarios y asesores  
3. Se describen beneficios del servicio  

### Flujos alternos  
- No aplica  

### Postcondiciones  
- El usuario comprende el valor del servicio de comisionistas  

### Reglas de negocio  
- RN-05: Los comisionistas no pueden registrarse desde la landing page
