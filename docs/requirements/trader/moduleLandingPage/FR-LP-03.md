## RF-LP-03 – Acceder al registro de usuario trader

### Descripción  
El sistema debe permitir al usuario no autenticado iniciar el proceso de registro como trader mediante un acceso desde la landing page.

### Actor(es)  
- Trader (no autenticado)

### Precondiciones  
- El usuario se encuentra en la landing page  

### Flujo básico  
1. El usuario selecciona la opción "Registrarse"  
2. El sistema redirige al formulario de registro de trader  

### Flujos alternos  
- No aplica  

### Postcondiciones  
- El usuario accede al flujo de registro  

### Reglas de negocio  
- RN-03: El registro disponible desde la landing es exclusivamente para traders
