## RF-LP-07 – Visualizar información de seguridad y confianza

### Descripción  
El sistema debe permitir al usuario visualizar información relacionada con la seguridad de la plataforma y la gestión de fondos.

### Actor(es)  
- Trader (no autenticado)

### Precondiciones  
- El usuario accede a la landing page  

### Flujo básico  
1. El sistema presenta información sobre seguridad  
2. Se describe el uso de autenticación multifactor  
3. Se informa sobre el uso de terceros para gestión de fondos  
4. Se menciona el enfoque en transparencia y trazabilidad  

### Flujos alternos  
- No aplica  

### Postcondiciones  
- El usuario percibe confianza en la plataforma  

### Reglas de negocio  
- RN-07: La información debe ser clara y comprensible para usuarios no expertos
