## RF-LP-06 – Visualizar información sobre suscripción premium

### Descripción  
El sistema debe permitir al usuario visualizar información sobre los beneficios del plan premium ofrecido por la plataforma.

### Actor(es)  
- Trader (no autenticado)

### Precondiciones  
- El usuario accede a la landing page  

### Flujo básico  
1. El sistema presenta información del plan premium  
2. Se muestran beneficios como alertas personalizadas y funcionalidades avanzadas  
3. Se muestra el costo del plan (mensual y anual)  

### Flujos alternos  
- No aplica  

### Postcondiciones  
- El usuario comprende el valor del plan premium  

### Reglas de negocio  
- RN-06: El costo de la suscripción debe mostrarse en USD
