# Requisitos Funcionales – Módulo Servicio de Comisionistas  
## Sistema: Nexus

---

## Descripción del Módulo

Este módulo permite al trader explorar comisionistas, analizar su perfil, y establecer contratos de asesoría u operación. Define la relación entre trader y comisionista, incluyendo estados del contrato, permisos y restricciones operativas.

---

## RF-CM-01 – Visualizar promoción del servicio de comisionistas

### Descripción  
El sistema debe presentar al trader información general sobre el servicio de comisionistas, incluyendo su funcionamiento, beneficios y tipos de interacción disponibles.

### Actor(es)  
- Trader  

### Precondiciones  
- El usuario ha completado la selección de plan  

### Flujo básico  
1. El sistema muestra información del servicio  
2. Se explica:
   - Asesoría (recomendaciones)  
   - Operación (ejecución directa)  
3. Se informa que las condiciones específicas se definen mediante contrato  

### Flujos alternos  
- N/A  

### Postcondiciones  
- El usuario comprende el funcionamiento del servicio
