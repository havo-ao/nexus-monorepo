## RF-CM-11 – Omitir contratación de comisionista

### Descripción  
El sistema debe permitir al trader omitir la contratación de un comisionista y continuar con el flujo del sistema.

### Actor(es)  
- Trader  

### Precondiciones  
- El usuario se encuentra en la sección de comisionistas  

### Flujo básico  
1. El usuario selecciona omitir  
2. El sistema redirige al dashboard principal  

### Flujos alternos  
- N/A  

### Postcondiciones  
- El usuario continúa sin comisionista
