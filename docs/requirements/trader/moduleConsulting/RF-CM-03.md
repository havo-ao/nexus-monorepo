# RF-CM-03 – Visualizar perfil básico de comisionista

### Descripción  
El sistema debe permitir al trader visualizar un perfil básico de un comisionista.

### Actor(es)  
- Trader  

### Precondiciones  
- El usuario selecciona un comisionista  

### Flujo básico  
1. El sistema muestra información:
   - Nombre  
   - Edad  
   - Identificación  
   - País  
   - Idiomas  
   - Experiencia  
   - Traders asesorados  
   - Drawdown  
   - Perfil de riesgo  
   - Calificación  
   - Tasa de retención  
   - Tiempo de respuesta  
   - Medio de comunicación  
   - Monto mínimo sugerido  
2. El sistema muestra enlace a portafolio completo externo  

### Flujos alternos  
- N/A  

### Postcondiciones  
- El usuario comprende el perfil del comisionista  

### Reglas de negocio  
- RN-CM-01: Las métricas son gestionadas exclusivamente por el administrador
