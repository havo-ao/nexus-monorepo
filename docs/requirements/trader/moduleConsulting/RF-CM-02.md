## RF-CM-02 – Visualizar listado de comisionistas

### Descripción  
El sistema debe permitir al trader visualizar un listado de comisionistas disponibles junto con filtros de búsqueda.

### Actor(es)  
- Trader  

### Precondiciones  
- El usuario accede a la sección de comisionistas  

### Flujo básico  
1. El sistema muestra listado de comisionistas  
2. El sistema permite aplicar filtros:
   - Costos  
   - Factor de riesgo  
   - Experiencia  
   - País  
   - Otros relevantes  
3. El sistema actualiza resultados según filtros  

### Flujos alternos  
- 2A. Sin resultados  
  - El sistema informa al usuario  

### Postcondiciones  
- El usuario identifica comisionistas de interés
