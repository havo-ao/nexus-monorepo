## RF-CM-04 – Visualizar medios de comunicación externos

### Descripción  
El sistema debe mostrar los medios de comunicación del comisionista cuando este ofrece únicamente servicios de asesoría.

### Actor(es)  
- Trader  

### Precondiciones  
- El comisionista ofrece solo asesoría  

### Flujo básico  
1. El sistema muestra medios de contacto externos  
2. El sistema informa que la comunicación se realiza fuera de la plataforma  

### Flujos alternos  
- N/A  

### Postcondiciones  
- El usuario conoce cómo comunicarse con el comisionista  

### Reglas de negocio  
- RN-CM-02: El sistema no gestiona comunicación directa entre usuarios
