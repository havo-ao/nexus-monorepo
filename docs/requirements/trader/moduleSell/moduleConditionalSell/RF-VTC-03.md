## RF-VTC-03 – Monitorear precio de mercado

### Descripción  
El sistema debe monitorear continuamente el precio de acciones con órdenes condicionadas.

### Actor(es)  
- Sistema  

### Flujo básico  

1. El sistema consulta precio de mercado  
2. Compara con el valor definido  
3. Si la condición se cumple:
   - ejecuta la venta
