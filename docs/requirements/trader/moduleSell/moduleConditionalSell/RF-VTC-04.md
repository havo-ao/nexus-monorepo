## RF-VTC-04 – Ejecutar venta condicionada

### Descripción  
El sistema debe ejecutar la venta cuando se alcance el precio definido.

### Actor(es)  
- Sistema  

### Flujo básico  

1. El sistema ejecuta la orden  
2. Actualiza portafolio  
3. Actualiza saldo del trader  
4. Calcula comisión  
5. Distribuye comisión  
6. Notifica al usuario
