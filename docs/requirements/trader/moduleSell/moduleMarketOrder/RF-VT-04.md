## RF-VT-04 – Preparar venta en mercado abierto
### Descripción  
El sistema debe calcular los valores de la operación antes de ejecutarla.

### Actor(es)  
- Sistema  

### Flujo básico  

1. El sistema obtiene precio actual  
2. Calcula:

- subtotal  
- comisión  
- total final  

3. El sistema muestra el detalle al trader
