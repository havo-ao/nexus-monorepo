## RF-VT-06 – Advertencia en mercado cerrado

### Descripción  
El sistema debe informar al trader cuando intenta vender acciones fuera de horario operativo.

### Actor(es)  
- Sistema  

### Flujo básico  

1. El sistema muestra mensaje:

"El mercado está cerrado. La orden se ejecutará a precio de apertura. El valor puede diferir del precio actual."

2. El usuario decide continuar o cancelar
