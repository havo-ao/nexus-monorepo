## RF-DB-02 – Visualizar listado general de acciones

### Descripción  
El sistema debe permitir visualizar un listado expandible de acciones disponibles en el mercado.

### Actor(es)  
- Trader  

### Flujo básico  
1. El sistema muestra listado de acciones  
2. El usuario puede expandir cada acción  
3. El sistema muestra opciones:
   - Ver detalle  
   - Comprar  
   - Vender  

### Flujos alternos  
- N/A  

### Postcondiciones  
- El usuario accede a acciones disponibles
