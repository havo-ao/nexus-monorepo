
## RF-PT-05 – Acceder a la venta de acciones

### Descripción  
El sistema debe permitir al trader iniciar el proceso de venta de una acción desde el portafolio.

### Actor(es)  
- Trader  

### Precondiciones  
- El usuario posee la acción  

### Flujo básico  
1. El usuario selecciona la acción  
2. El sistema muestra opción "Vender"  
3. El sistema redirige al módulo de venta  

### Postcondiciones  
- Se inicia el flujo de venta de acciones
