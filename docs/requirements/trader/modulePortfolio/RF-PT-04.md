## RF-PT-04 – Actualizar valor de acciones

### Descripción  
El sistema debe actualizar el valor de mercado de las acciones en el portafolio.

### Actor(es)  
- Sistema  

### Flujo básico  
1. El sistema consulta datos de mercado  
2. El sistema actualiza precios actuales  
3. El sistema recalcula métricas de rendimiento  

### Reglas de negocio  
- RN-PT-02: Los datos deben provenir de la fuente de mercado integrada
