# Requisitos Funcionales – Sección Gestión de Comisionistas (Trader)  
## Sistema: Nexus

---

## Descripción del Módulo

Este módulo permite al trader gestionar la relación con comisionistas, incluyendo visualización de contratos activos, historial y aprobación de recomendaciones.

---

## RF-CMT-01 – Visualizar página principal de comisionistas

### Descripción  
El sistema debe mostrar el listado de comisionistas disponible.

### Flujo básico  
1. El sistema presenta listado y filtros  
2. Permite navegación a perfiles
