## RF-CMT-06 – Gestionar notificaciones de comisionista

### Descripción  
El sistema debe permitir al trader gestionar recomendaciones del comisionista.

### Flujo básico  
1. El sistema muestra buzón de recomendaciones  
2. El trader selecciona una recomendación  
3. El trader aprueba o rechaza  

### Flujos alternos  
- 3A. Rechazo  
  - No se ejecuta operación  

### Reglas de negocio  
- RN-CMT-04: Solo aplica para contratos de asesoría
