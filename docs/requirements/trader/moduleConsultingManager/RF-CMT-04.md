## RF-CMT-04 – Generar reporte de contrato

### Descripción  
El sistema debe permitir descargar un reporte detallado de un contrato.

### Flujo básico  
1. El usuario solicita descarga  
2. El sistema genera reporte con:
   - Ganancias y pérdidas  
   - Acciones operadas  
   - Cantidades  
   - Valores  
   - Fechas  
   - Comisiones
