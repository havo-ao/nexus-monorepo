## RF-CR-04 – Visualizar historial de transacciones

### Descripción
El sistema debe permitir al trader consultar el historial de movimientos de la cartera.

### Actor(es)
- Trader

### Flujo básico

1. El sistema consulta las transacciones registradas
2. El sistema muestra para cada movimiento:
   - tipo de operación
   - monto
   - fecha
   - estado de la operación

### Postcondiciones
- El usuario puede revisar sus movimientos financieros
