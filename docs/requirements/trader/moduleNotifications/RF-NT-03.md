## RF-NT-03 – Gestionar notificaciones de seguridad

### Descripción  
El sistema debe permitir activar o desactivar notificaciones relacionadas con seguridad.

### Actor(es)  
- Trader  

### Precondiciones  
- Usuario autenticado  

### Flujo básico  
1. El sistema muestra opciones:
   - Inicio de sesión  
   - Cambio de datos personales  
   - Cambio de datos financieros  
2. El usuario activa/desactiva cada opción  
3. El sistema guarda configuración  

### Flujos alternos  
- N/A  

### Postcondiciones  
- Configuración actualizada
