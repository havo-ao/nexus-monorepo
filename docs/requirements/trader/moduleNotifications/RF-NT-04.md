## RF-NT-04 – Gestionar notificaciones de suscripción

### Descripción  
El sistema debe permitir activar o desactivar notificaciones relacionadas con el estado del plan.

### Actor(es)  
- Trader  

### Flujo básico  
1. El sistema muestra opciones:
   - Próxima expiración  
   - Downgrade  
   - Renovación  
2. El usuario configura preferencias  
3. El sistema guarda configuración
