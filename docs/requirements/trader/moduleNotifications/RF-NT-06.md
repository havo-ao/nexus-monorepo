## RF-NT-06 – Gestionar notificaciones de validación

### Descripción  
El sistema debe permitir activar o desactivar notificaciones relacionadas con el estado de validación financiera.

### Actor(es)  
- Trader  

### Flujo básico  
1. El sistema muestra estados:
   - En validación  
   - Aprobado  
   - Rechazado  
   - Error técnico  
2. El usuario configura preferencias  
3. El sistema guarda configuración
