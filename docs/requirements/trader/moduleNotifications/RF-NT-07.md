## RF-NT-07 – Gestionar notificaciones de comisionistas

### Descripción  
El sistema debe permitir activar o desactivar notificaciones relacionadas con contratos y actividad de comisionistas.

### Actor(es)  
- Trader  

### Flujo básico  
1. El sistema muestra opciones:
   - Aprobación de contrato  
   - Rechazo de contrato  
   - Inicio de contrato  
   - Finalización o cancelación  
   - Recomendaciones de asesoría  
2. El usuario configura preferencias  
3. El sistema guarda configuración
