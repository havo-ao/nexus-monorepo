# Requisitos Funcionales – Compra de Acciones (Market Order)  
## Sistema: Nexus

---

## Descripción del Módulo

Este módulo permite al trader realizar compras de acciones mediante órdenes de mercado, tanto en horario operativo como fuera de este.

---

## RF-CP-01 – Seleccionar tipo de orden de compra

### Descripción  
El sistema debe permitir al trader seleccionar el tipo de orden de compra.

### Actor(es)  
- Trader  

### Precondiciones  
- El usuario accede a la sección de compra  

### Flujo básico  
1. El sistema muestra opciones:
   - Market Order  
   - Limit Order  
2. El usuario selecciona Market Order
