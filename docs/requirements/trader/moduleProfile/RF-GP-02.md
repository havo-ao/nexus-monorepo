## RF-GP-02 – Editar datos personales

### Descripción  
El sistema debe permitir al trader modificar su información personal.

### Flujo básico  
1. El usuario edita datos  
2. El sistema valida formato  
3. El sistema guarda cambios  

### Flujos alternos  
- 2A. Datos inválidos  
  - Solicita corrección  

### Reglas de negocio  
- RN-GP-02: Cambios en correo o teléfono requieren revalidación  
- RN-GP-03: Se debe notificar cualquier cambio
