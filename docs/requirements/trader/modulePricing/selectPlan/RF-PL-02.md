## RF-PL-02 – Seleccionar plan

### Descripción  
El sistema debe permitir al trader seleccionar un plan (Free o Premium).

### Actor(es)  
- Trader  

### Precondiciones  
- El usuario se encuentra en la pantalla de selección de plan  

### Flujo básico  
1. El usuario selecciona un plan  
2. El sistema registra la selección  

### Flujos alternos  
- N/A  

### Postcondiciones  
- El plan seleccionado queda asociado al usuario  

### Reglas de negocio  
- RN-PL-03: El usuario puede omitir la selección de plan Premium y continuar con el plan Free
