## RF-PL-01 – Visualizar planes disponibles

### Descripción  
El sistema debe permitir al trader visualizar los planes disponibles (Free y Premium), incluyendo sus beneficios y costos asociados.

### Actor(es)  
- Trader  

### Precondiciones  
- El usuario ha completado el proceso de registro y verificación  

### Flujo básico  
1. El sistema muestra los planes disponibles:
   - Plan Free  
   - Plan Premium  
2. El sistema presenta los beneficios del plan Premium:
   - Alertas personalizadas  
   - Watchlist  
   - Seguridad adicional  
   - Estadísticas avanzadas  
3. El sistema muestra los costos del plan Premium:
   - USD $12 mensual  
   - USD $120 anual  
4. El plan Free se presenta como opción por defecto  

### Flujos alternos  
- N/A  

### Postcondiciones  
- El usuario comprende las diferencias entre planes  

### Reglas de negocio  
- RN-PL-01: El plan Free debe estar disponible sin costo  
- RN-PL-02: Los precios deben mostrarse en USD
