## PRM-FR-02 – Gestionar watchlist

### Descripción  
El sistema debe permitir al trader gestionar una lista personalizada de acciones visibles en su dashboard.

### Actor(es)  
- Trader Premium  

### Precondiciones  
- El usuario posee plan Premium activo  

### Flujo básico  
1. El usuario agrega acciones a la watchlist  
2. El usuario elimina acciones de la watchlist  
3. El sistema guarda los cambios  
4. El sistema muestra la watchlist en el dashboard  

### Flujos alternos  
- N/A  

### Postcondiciones  
- La watchlist queda configurada  

### Reglas de negocio  
- RN-PRM-02: En el plan Free las acciones mostradas son aleatorias
