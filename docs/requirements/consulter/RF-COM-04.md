## RF-COM-04 – Consultar cotización actual de acciones

### Descripción
El sistema debe permitir al comisionista consultar la información actual del mercado de acciones.

### Actor(es
- Comisionista

### Precondiciones
- Usuario autenticado.

### Flujo básico

1. El comisionista accede a la sección de consulta de mercado.
2. El sistema consulta los datos actuales del mercado.
3. El sistema muestra para cada acción:
   - símbolo
   - precio actual
   - precio bid
   - precio ask
   - volumen de negociación
   - mercado o exchange.
